FAQ — Session Player (MeeZen)

What is Sleep Timer for? ▾

Sleep Timer stops playback after the number of minutes you set if the session is still playing.

Example (cuts before the end):
A 60-minute session + Sleep Timer = 20 minutes
→ Playback stops after 20 minutes, even if the session hasn’t finished.

✅ Best for: falling asleep without audio running all night.
⚠️ Note: Sleep Timer does not loop the session.

What is Play For for? ▾

Play For keeps the session playing for a set duration (in minutes). If the session ends before that time, it will restart automatically until the timer finishes.

Example:
A 3-minute session + Play For = 15 minutes
→ The session will replay as needed and run for about 15 minutes.

✅ Best for: extending short sessions without manually restarting.

What is Repeat for? ▾

Repeat replays the session a specific number of times.

Example:
Repeat = 9×
→ The session will play 9 times.

✅ Best for: repeating affirmations or subliminals a fixed number of times.

What happens if I enable more than one mode? ▾

MeeZen uses this simple priority order:
Sleep Timer > Play For > Repeat

  • If Sleep Timer is on: it stops everything when its timer ends.
  • If Sleep Timer is off: Play For runs until its minutes end.
  • If both are off: Repeat plays the selected number of times.

Example:
Repeat 10× + Play For 20 min + Sleep Timer 12 min
→ Playback stops after 12 minutes.

What is Listening Style for? ▾

Listening Style helps match sound intensity to your situation.
(Reminder: adjust your device volume first, then choose a Listening Style.)

  • Very Soft / Soft: ideal for sleep or discreet listening
  • Calm / Standard: for normal listening

✅ Important: Listening Style only changes loudness — it does not change duration.

Is Listening Style useful for subliminals? ▾

Yes — subliminals work best when they’re subtle and in the background, not loud or distracting.

Recommended setup:

  1. Set your device volume to a comfortable overall level.
  2. Choose a Listening Style:
    • Best for subliminals: Very Soft (or Soft)
  3. Keep the audio barely noticeable as background sound.

⚠️ Note: Listening Style only affects loudness, not duration or repeats.

What do Off and Unlimited mean? ▾
  • Off: the option is disabled.
  • Unlimited: playback continues with no limit (until you stop it).
